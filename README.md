Você está buscando APIs públicas e gratuitas (ou com planos gratuitos) para praticar suas habilidades em desenvolvimento web ou mobile? Este repositório reúne **30 APIs grátis, públicas ou com plano gratuito disponível**, organizadas por categoria para te ajudar a criar projetos incríveis!

> ⚠️ **Atenção:** Muitas dessas APIs oferecem planos gratuitos com limitações — como número de requisições por dia ou necessidade de cadastro e chave API.  
> Sempre confira as condições diretamente na documentação oficial antes de usar.

## 🚀 Lista de APIs

### 1. OpenStreetMap API
- 🔎 Dados Abertos / Geoespaciais  
- **URL:** https://www.openstreetmap.org/api  
- **Plano Gratuito:** ✅ Sim, totalmente gratuito  
- **Descrição:** Dados geoespaciais livres para criação de mapas personalizados.

### 2. NASA API
- 🔎 Dados Públicos / Espaço  
- **URL:** https://api.nasa.gov/  
- **Plano Gratuito:** ✅ Sim, requer cadastro para chave gratuita  
- **Descrição:** Dados e imagens espaciais, missões, astronomia.

### 3. World Bank API
- 📊 Dados Financeiros / Econômicos  
- **URL:** https://datahelpdesk.worldbank.org/knowledgebase/articles/889386-developer-information-overview  
- **Plano Gratuito:** ✅ Sim, público e aberto  
- **Descrição:** Dados econômicos e sociais globais.

### 4. GeoNames API
- 🔎 Dados Geográficos  
- **URL:** https://www.geonames.org/export/web-services.html  
- **Plano Gratuito:** ✅ Sim, com limite de requisições por hora  
- **Descrição:** Informações geográficas e nomes de lugares.

### 5. Open Library API
- 🔎 Dados Educacionais / Livros  
- **URL:** https://openlibrary.org/developers/api  
- **Plano Gratuito:** ✅ Sim, aberto  
- **Descrição:** Dados de livros, autores e capas.

### 6. OpenWeatherMap API
- 🌤️ Clima  
- **URL:** https://openweathermap.org/api  
- **Plano Gratuito:** ✅ Sim, plano free limitado (até 60 chamadas/minuto)  
- **Descrição:** Dados de clima atuais, previsão e históricos.

### 7. WeatherAPI.com
- 🌤️ Clima  
- **URL:** https://www.weatherapi.com/  
- **Plano Gratuito:** ✅ Sim, plano free com limite diário  
- **Descrição:** Clima, alertas e previsões.

### 8. StormGlass API
- 🌊 Dados Oceânicos / Clima  
- **URL:** https://stormglass.io/  
- **Plano Gratuito:** ✅ Sim, plano free com limitações  
- **Descrição:** Dados oceânicos e atmosféricos para apps marítimos.

### 9. Visual Crossing Weather API
- 🌤️ Clima  
- **URL:** https://www.visualcrossing.com/weather-api  
- **Plano Gratuito:** ✅ Sim, plano free limitado  
- **Descrição:** Dados meteorológicos históricos e previsão.

### 10. Weatherbit API
- 🌤️ Clima  
- **URL:** https://www.weatherbit.io/api

- **Descrição:** Dados meteorológicos e qualidade do ar.
